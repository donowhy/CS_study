# Switch
    네트워크 자체 == Switch
        ex) 라우터, F/W, IPS - 보안 스위칭 = 경로 선정

        ex) peer, server, client

    IP 주소로 Switch시 L3 스위치
    L3 스위치중 가장 대표적인것 = 라우터
