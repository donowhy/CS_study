# IP(Internet Protoco) 주소
    HOST에 대한 식별자
    대한민국의 행정체계(Network)에서 개인에 대한 식별자 = 주민등록번호(IP 주소)

    주소길이
    IPv4 = 32비트 = 2**32 = 43억
    IPv6 = 128비트

    인터넷 호스트 제한 = 43억
    전세계 컴퓨터는 43억개 초과

    192.168.60.14 = 8+8+8+8
    0~255
    IP주소는 8비트씩 끊어서 표시

    192는 2진수로 1100 0000    
         16진수로 C0
    
    IP주소는 크게 2로 나뉨
    ~.~.~ = 네트워크 ID = 24비트
        네트워크 ID의 길이를 나타내는게 넷마스크
    .~ = 호스트 ID = 8비트
